## Diferencias C vs Python

| Aspecto | C | Python |
|---------|---|--------|
| Tipo | `int x = 5;` | `x = 5` |
| Print | `printf("%d", x);` | `print(x)` |
| Input | `scanf("%d", &x);` | `x = int(input())` |
| Loop | `for(i=0; i<5; i++)` | `for i in range(5):` |
| Array | `int arr[] = {1,2,3};` | `arr = [1,2,3]` |
